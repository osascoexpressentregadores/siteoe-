## Como publicar

1. Extraia o ZIP.
2. Suba o conteúdo extraído direto na raiz do repositório `siteoepremiun`.
3. A raiz do repositório precisa mostrar `index.html`, `assets/`, `agenda/`, `servicos/`, `regioes/`, `entregadores/`, `sitemap.xml` e `robots.txt`.
4. Não suba uma pasta por dentro do repositório.
5. Depois de publicar, abra em aba anônima ou use Ctrl + F5.
